# WAV → CSV Converter (Streamlit App)

このアプリは WAV ファイルを読み込み、指定したサンプリング周波数（既定 1000 Hz）に
リサンプリングして CSV 形式でダウンロードできる Web アプリです。  
Streamlit Cloud 上で動作し、csvファイルをローカルに保存できます。

---

## 🚀 使い方

1. **サイドバーから WAV ファイルをアップロード**
   - 200 MB 超のファイルは警告表示（処理は続行されます）

2. **サンプリング周波数を指定**
   - 範囲：100 – 2000 Hz（既定 1000 Hz）

3. **「リサンプル」ボタンを押して変換**
   - Kaiser β = 8.6 の polyphase フィルタで補間
   - 元データと変換後データを時系列グラフで表示
   - グラフは全体で最大 20 万点に自動ダウンサンプルして描画

4. **「CSVをダウンロード」ボタンで保存**
   - クライアントPC（ブラウザ側）に CSV ファイルを保存
   - CSV の形式：
     ```
     ch0, ch1, ...
     0.001234, 0.002345, ...
     ```
   - 文字コード UTF-8、小数6桁、インデックス列なし
